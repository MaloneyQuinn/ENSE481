void pid_init ( pid * controller)

Initializes values of pid struct for use.

sets pid constants (kp, ki, kd, time interval) to their values as well as setting all other info to 0. kp, ki, and kd all tuned through testing.

Parameters
controller(*pid) pid struct that is getting initialized gets passed in.

Takes current value and adjust duty cycle to balance ping pong ball.
